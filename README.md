## About

The first bootcamp project in the HTML section of the Foundation Course from The Odin Project focuses on creating a basic webpage to apply the fundamental concepts of HTML. Students are tasked with structuring an HTML document that includes essential elements such as headings, paragraphs, bulleted and numbered lists, internal and external links, and image insertion. 

The primary goal of the project is to provide hands-on experience in building web pages, allowing participants to experiment with HTML structure and semantics. Through this task, students gain confidence and competence in using HTML tags and attributes, laying a solid foundation for more advanced projects and future learning of CSS and JavaScript.
